# LuBan.Workflow

> **Author**: yswenli | **Contact**: yswenli@outlook.com | **Repository**: [https://github.com/yswenli/luban-framework](https://github.com/yswenli/luban-framework)

> Workflow engine component (planned)

This module is currently in the planning phase and does not yet contain implementation code.

The approval flow capabilities of LuBan.Framework are provided by **LuBan.ApprovalFlow**, which includes:

- Graph-theory model workflow definitions (JSON configuration)
- Conditional gateway routing (rule engine)
- Countersign aggregation (AND / OR / ratio)
- Task delegation and routing
- HTTP callback nodes
- Event-driven notifications (integrated with EventBus)

For approval flow functionality, please use `LuBan.ApprovalFlow` directly.
