# LuBan.Workflow

> **作者**: yswenli | **代码仓库**: [https://github.com/yswenli/luban-framework](https://github.com/yswenli/luban-framework)

> 工作流引擎组件（规划中）

该模块目前处于规划阶段，尚未包含实现代码。

LuBan.Framework 的审批流能力已由 **LuBan.ApprovalFlow** 提供，包含：

- 图论模型流程定义（JSON 配置）
- 条件网关路由（规则引擎）
- 会签聚合（AND / OR / 比例）
- 任务委派与流转
- HTTP 回调节点
- 事件驱动通知（集成 EventBus）

如需审批流功能，请直接使用 `LuBan.ApprovalFlow`。
